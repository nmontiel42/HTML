Etiqueta de cierre. [ESTRUCTURA DE UN ELEMENTO HTML] Los elementos no son etiquetas, estan representados por ellas. Algunos elementos estan compuestos por una sola etiqueta que se usa para insertar/incorporar algo dentro del documento. (Etiquetas independientes). [ATRIBUTOS EN HTML] Definen valores po propiedades que utilizan los navegadores para procesar el documento. Link Rel identifica la relacion entre el documento actual y el de destino. [TAG ANCLA] Los atributos son parejas clave-valor separados por el caracter "=". Los atributos globales estan disponibles para todos los elementos. La mayoria de los elementos tienen un conjunto especifico de atributos, disponibles solo para ellos. [TAG GENERICOS] Las etiquetas genericas son etquetas neutrales que se utilizan con el unico proposito de separar un contenido de otro. Son una especie de macrocontenedores en los que insertar nuestros elementos, dividiendolos. La etiqueta
puede contener parrafos, imagenes, titulos, etc. Tiene un comportamiento de bloqueo. La etiqueta marca una parte del texto o de un documento, al que queremos darle una caracteristica en particular. El intervalo es una etiqueta online, lo que significa que no tiene el comportamiento de bloqueo.
